# Changelog

All notable public changes to EmoC will be documented in this file.

The format follows a simple date-based release log. Public versions should match
the version in `pubspec.yaml`.

## [1.0.0] - 2026-07-02

### Added

- Mobile-first Android UI built with Flutter.
- Native Kotlin integration for media playback, media session, and desktop
  lyrics overlay.
- NetEase Cloud Music login/session flow.
- Home recommendations, playlist library, search, playback controls, and lyrics
  page.
- Desktop lyrics settings including opacity, color, size, lock, multi-line mode,
  and foreground auto-hide.
- Dynamic color, light/dark/system theme settings, cache clearing, and playback
  quality preference.
- Signed release build configuration.
- Public documentation, privacy policy, disclaimer, versioning rules, and
  GPL-3.0-or-later license.

### Notes

- EmoC is third-party, free, and unofficial.
- Service availability depends on NetEase Cloud Music.
